% Index
Falls gewünscht kann hier ein Index erstellt werden. (optional)

Dazu müssen im Text alle Wörter, die im Index auftauchen sollen mit dem Befehl 
\begin{verbatim}
\index{Befehl}
\end{verbatim}
für den Index markiert werden.

Der Index wird dann mit
\begin{verbatim}
\printindex
\end{verbatim}
erstellt werden.

Weitere Informationen bezüglich Indexerstellung sind in der beigefügten Datei "`/dokumente/makeidx.pdf"' enthalten.

Das fertige Indexverzeichnis sieht dann so aus: (siehe Folgeseite)
\printindex
